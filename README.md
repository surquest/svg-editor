# svg-editor
SVG editor with code as well as canva base editing

## Live Demo

🚀 **[Try it live on GitHub Pages](https://surquest.github.io/svg-editor/)**

## Features

- **Code Editor** — Edit SVG code directly with real-time preview
- **Live Preview** — Instantly see your SVG as you type
- **Shape Library** — Quickly insert common shapes (rect, circle, ellipse, line, text, path)
- **Import / Export** — Import SVG files or download your work
- **Copy to Clipboard** — Copy the SVG source with one click
- **Validation** — Instant feedback on SVG syntax errors

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project is automatically deployed to [GitHub Pages](https://surquest.github.io/svg-editor/) via GitHub Actions on every push to the `main` branch.
